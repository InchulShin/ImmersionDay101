# 6. 추가 실습

추가 실습은 아래의 내용으로 구성되어 있습니다. 각 실습은 앞선 실습과는 달리 유기적으로 연결되어 있지 않기 때문에 원하는 부분만 실습할 수 있습니다.

추가 실습의 경우, 오늘 실습한 자원을 삭제하시면 진행할 수 없습니다. 따라서, 추가 실습 진행 후, 일괄적으로 자원을 삭제하면 됩니다.
